Este projeto foi desenvolvido como uma atividade proposta pelo professor Carlos Alberto, com o objetivo de praticar o trabalho em equipe utilizando Git e GitHub.

A proposta consiste em criar um site principal que reúne links individuais de cada integrante do grupo. Cada link redireciona o usuário para o site pessoal que cada aluno desenvolveu no ano anterior, permitindo navegar entre os projetos de forma organizada.

O foco da atividade é o uso colaborativo de versionamento de código, organização de repositório e integração entre os membros do grupo.
